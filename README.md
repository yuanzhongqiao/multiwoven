<p dir="auto">Multiwoven is a monorepo that consists of three main services:</p>
<ul dir="auto">
<li>
<p dir="auto"><a href="https://github.com/Multiwoven/multiwoven/tree/main/server">server</a> - The backend service that acts as a control plane for managing data sources, models, and syncs.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/Multiwoven/multiwoven/tree/main/ui">ui</a> - The frontend react application that provides a user interface to manage data sources, destinations, and confgure syncs.</p>
</li>
<li>
<p dir="auto"><a href="https://github.com/Multiwoven/multiwoven/tree/main/integrations">integrations</a> - A Ruby Gem that provides a framework to build connectors to support a wide range of data sources and destinations.</p>
</li>
</ul>
